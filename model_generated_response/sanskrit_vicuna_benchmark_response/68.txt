Instruction in English:  Find the value of f(2) given f(x) = 5x^3 - 2x + 3..
                        Response in English: To find the value of f(2), we need to substitute 2 for x in the function f(x) = 5x^3 - 2x + 3.

f(2) = 5(2)^3 - 2(2) + 3
f(2) = 5(8) - 4 + 3
f(2) = 40 - 4 + 3
f(2) = 39

Therefore, the value of f(2) is 39..
